Battleship (Windows Forms Application)

This project is a Windows Forms desktop game based on the classic Battleship game where players try to find and sink hidden ships on a grid.

📖 About

A simple Windows Forms Battleship game built in C# where players compete to locate and destroy enemy ships on a grid.

🚀 Features
Grid-based gameplay
Player vs Computer mode
Ship placement system
Hit / Miss detection
Turn-based logic
Simple Windows Forms UI

Battleship
│
├── Forms
├── Models

⚙️ Tech Stack
C#

Open project in Visual Studio
Build solution
Run (F5)
